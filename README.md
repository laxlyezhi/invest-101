# invest-101

> 面向普通人的投资入门课——从金融第一性原理出发，用类比与案例拆解三张财报、现金流和估值，帮你建立可独立验证的投资决策框架。

## 课程结构

6 周 · 30 天 · 每天一个核心概念

| 周 | 主题 | 核心概念 |
|---|------|---------|
| W1 | 金融的第一性原理 | 时间价值 · 复利 · 风险溢价 |
| W2 | 资产负债表 | A = L + E · 流动性 · 资产质量 · 生意模式 DNA |
| W3 | 利润表 | 从营收到净利润，每一行都可能造假 |
| W4 | 现金流量表 | 利润是观点，现金是事实 |
| W5 | 三表勾稽 + 杜邦分析 | 把三张表拼成一个动态系统 |
| W6 | 基础估值 | PE · PB · DCF · 安全边际 |

## 设计理念

- **第一性原理驱动** — 不堆术语，从"钱为什么有时间价值"讲起，每个概念追溯到最底层的逻辑
- **类比先行** — 用物理学/工程学的直觉映射金融概念（折现率 = 系统特征频率，资产负债表 = 状态变量）
- **案例实打实** — 2020 美联储降息、特斯拉估值、康美药业造假…每个概念配真实案例
- **AI 助手旁听** — 内置 AI 对话，随时拆概念、换例子、回答学习中的困惑

## 技术栈

- **Alpine.js** — 轻量响应式框架
- **Tailwind CSS** — 原子化样式
- **KaTeX** — 数学公式渲染
- **marked.js** — Markdown 解析
- 纯静态，零构建步骤，打开 `index.html` 即可运行

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/yezhi/invest-101.git
cd invest-101

# 直接用浏览器打开
open index.html

# 或启动本地服务器
python3 -m http.server 8765
# 访问 http://localhost:8765
```
